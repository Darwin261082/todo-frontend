Sistema de Gestión de Tareas

Descripción del Proyecto

Este proyecto consiste en el desarrollo de una aplicación web para la gestión de tareas (To-Do List). El sistema permite a los usuarios registrarse, iniciar sesión y administrar sus tareas de manera sencilla mediante una interfaz web interactiva.

Frontend

El frontend fue desarrollado utilizando Angular, un framework moderno para la construcción de aplicaciones web dinámicas. La interfaz permite la interacción con el sistema mediante formularios, visualización de tareas y consumo de servicios HTTP para comunicarse con el backend.

Backend

El backend está desarrollado como una API REST que gestiona la autenticación de usuarios y la administración de tareas. La comunicación entre el frontend y el backend se realiza mediante peticiones HTTP utilizando formato JSON.

La arquitectura del backend está basada en el modelo de Arquitectura Hexagonal (Ports and Adapters), lo que permite separar la lógica de negocio de la infraestructura, facilitando el mantenimiento, escalabilidad y pruebas del sistema.

Funcionalidades

  • Registro de usuarios
  • Inicio de sesión con autenticación mediante token
  • Creación de tareas
  • Visualización de tareas
  • Edición y actualización del estado de las tareas
